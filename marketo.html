Getting Started

Installation & Setup

Follow these steps to set up and run the e-commerce app on your local machine:

Please make sure you have Flutter installed on your system. Follow the installation guide at Flutter.dev.

  1. Install an editor like VS Code or Android Studio
  2. Set up an Android emulator or connect a physical device
  3. Clone the repository
  4. Navigate to the project directory
  5. Fetch the dependencies by running flutter pub get
  6. Run the app on your connected device or emulator with flutter run

Adding New Languages

You can add a new language by following these steps or check the todo list in the project:

  1. Add a new JSON file in the assets/lang folder with the language code as the filename (e.g., fr.json for French)
  2. Add the translations in the JSON file in the following format:
{
  "key": "value",
  "key2": "value2"
}

You can use en.json as a reference.

  1. Update the supported locales list in the main.dart and my_app files
  2. Add the new language to the dropdown by updating the language list in lib/core/enum/enums.dart

Backend API Documentation

API Architecture:

  • RESTful API built with Node.js/Express
  • MongoDB database for data storage
  • JWT authentication
  • MVC architecture pattern

Main Endpoints:

# Authentication
POST   /api/auth/register
POST   /api/auth/login

# Users
GET    /api/users
GET    /api/users/:id
PUT    /api/users/:id
DELETE /api/users/:id

# Products
GET    /api/products
POST   /api/products
GET    /api/products/:id
PUT    /api/products/:id
DELETE /api/products/:id

# Orders
GET    /api/orders
POST   /api/orders
GET    /api/orders/:id
PUT    /api/orders/:id
